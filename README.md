# SplitCash

SplitCash is a bill splitting Android application built using Jetpack Compose and MVVM architecture.

The app allows users to:

- Enter bill amount and tip
- Add participants
- Track how much each person paid
- Calculate equal share
- Determine settlements (who owes whom)
- Save and view expense history

## Tech Stack

- Kotlin
- Jetpack Compose
- MVVM Architecture
- Navigation Compose
- Material 3

## Features

- Dynamic participant list
- Settlement calculation algorithm
- Expense history tracking
- Modern Compose UI

## Future Improvements

- Persistent storage with Room database
- Group-based expenses
- Expense editing
- UI/UX enhancements
